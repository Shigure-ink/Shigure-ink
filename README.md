# 关于“时雨ink”
## 关于名字
时雨（しぐれ）是日本俳句中转瞬即逝的灵感之雨\
所以时雨ink就是墨水与灵感的骤雨，当然，嫌绕口的话，你也可以说成时雨墨水，意思是一样的。
